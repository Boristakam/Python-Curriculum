# 🧬 Scientific Computing with Python
## Data Analysis | Algorithms | Mathematical Modelling

This repository contains my implementations and certification projects from the Scientific Computing with Python curriculum. It covers the transition from core Python syntax to complex algorithmic problem-solving and automated data processing.

### 🚀 Certification Projects
These represent the "capstone" challenges that require end-to-end logic design:
- Arithmetic Formatter: A tool to arrange and solve arithmetic problems vertically, focusing on string manipulation and formatting logic.
- Time Calculator: An algorithm to calculate durations and dates, handling edge cases like multi-day rollovers and leap years.
- Budget App: An Object-Oriented (OOP) system for category management, demonstrating class structures, method overriding, and data visualisation.
- Polygon Area Calculator: A geometric modelling tool using inheritance and polymorphism.
- Probability Calculator: A Monte Carlo simulation to determine the probability of drawing specific balls from a hat, showcasing randomisation and large-scale iteration.

### 🛠️ Technical Competencies
Throughout this course, I am refining the following engineering skills:
- Data Structures: Advanced use of Lists, Dictionaries, and Tuples for efficient data retrieval.
- Regular Expressions (Regex): Complex pattern matching for data parsing and validation.
- Object-Oriented Programming (OOP): Designing reusable, modular code using Classes and Inheritance.
- Numerical Logic: Implementing mathematical algorithms without external libraries to ensure a "first-principles" understanding of computation.
